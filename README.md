### üó∫Ô∏è Enhanced RME (Remere's Map Editor) - Now accepting contributions!

Modern fork with quality-of-life improvements for OpenTibia map creation. **This is my first public project and contributions are welcome!**

**Features:**
- Larger palette icons - Optional, configurable in Preferences
- Dark mode - Partially implemented, configurable in Preferences
- Right-click Replace Items - No need to manually enter item IDs
- Smart Replace scope - Choose between visible screen (same floor) or entire OTBM map
- Teleport coordinates display - Visual squares when placing teleports, useful for ActionID/UniqueID tracking
- Direct minimap export - Export to minimap.otmm directly from editor to your in-game minimap
- Wide client compatibility - Supports clients 7.4-12.86
- Configurable interface - All features are optional via Preferences/Interface
